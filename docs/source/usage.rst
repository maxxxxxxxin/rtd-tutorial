Usage
=====

.. _installation:

Installation
------------

To use Lumache, first install it using pip:

.. code-block:: console

    (.venv) $ pip install TriTan

Basic usage
-----------

Import TriTan as:

.. code-block:: python

   import TriTan

Initialise a TriTan model:

.. code-block:: python

    tritan = TriTan.TriTan()

Tran the model:

.. code-block:: python

    tritan.fit(mdata)
